# Cheatsheet
| Symbol | Meaning |
| --- | --- |
| . | Any character except new line |
| \w | Word |
| \W | Not Word |
| \d | Digit |
| \D | Not Digit |
| \s | Whitespace |
| \S | Not Whitespace |
| [abc] | Any of a,b,c |
| [^abc] | Not any of a,b,c |
| [a-z] | Character from a to z |
| ^abc | Start of the string |
| abc$ | End of the string |
| \b | Start of the string |
